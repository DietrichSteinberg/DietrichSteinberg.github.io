Coded Graphics - Spring 2022

Random Graphics

This project is a display of multiple graphics functioning at the same time.

Inosuke Head Pat

This project displays a graphic that uses a mouse pointer as a trigger and the hover function to activate a change in the graphic. The graphics are inspired by a meme of an anime character.

Spinning Levi

This project is meant to display a rotation function. The project displays a reference from an anime that I like.

Clock Project

This project was a creation of visual effects based on a timer or clock. The visual effects represent the muscles that are being engaged during multiple workout sessions on a day by day basis.

Galaxy Pinball

This graphic displays the bounce function. I dicided to have the theme be space, because I thought it would elivate the piece and make it more interesting.

Squid

This project uses mouse activity as a function trigger that gives the squid an agitated response. Was made using multiple seperate tabs in processing for personal organization reasons.

Stormy Night

For this project I combined the wave and noise function in order to make a landscape image with a cloady night sky. This graphic also displays the usage of layering to create a foreground and background.

Paint Brush

This project uses meme images to create a brush and eraser function based on the motion of a mouse. The brushes are activates by pressing down on the mouse. It also contains a function that uses number keys to swap between brushes.
